# College Staff Administration Portal

Frontend - Angular version 6.0.8

Controller - Python Flask RESTful library

Backend - MySQL

## API's


`1. Departments`

    http://localhost:5000/api/v1.0/protected/dept
    
    methods: GET, POST, PUT


`2. HOD`
